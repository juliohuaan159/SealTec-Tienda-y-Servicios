# SealTec-Tienda-y-Servicios
SealTec | Tienda y Servicios es una empresa especializada en soluciones informáticas, dedicada a la venta de equipos tecnológicos, accesorios, periféricos y suministros de impresión. Además, brinda servicios de soporte técnico, mantenimiento y reparación de computadoras,laptops e impresoras,asesoría para la implementación de soluciones tecnológica.
